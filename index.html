The Tetris Game

Instruções do Jogo

Bem-vindo ao Tetris! 🎮

  • Seta para a Esquerda (←): Move a peça para a esquerda.
  • Seta para a Direita (→): Move a peça para a direita.
  • Seta para Baixo (↓): Faz a peça descer mais rapidamente.
  • Seta para Cima (↑): Roda a peça no sentido horário.

O objetivo é encaixar as peças para completar linhas horizontais.

Cada linha eliminada aumenta a sua pontuação.

Não deixe as peças acumularem até ao topo!

Divirta-se e boa sorte! 🚀
